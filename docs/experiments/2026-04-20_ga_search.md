# 2026-04-20 GA Search

## Goal

Start a genetic algorithm that takes the strongest pairwise expressions and tries to improve them into a better expression.

The first GA run used the best pair from the pairwise screening:

- `feature_a = sex_unit`
- `feature_b = age_unit`

## Implementation

- [ga_best_expression.rs](/home/tarstars/prj/titanic_mle/rust/eml_tree_search/src/bin/ga_best_expression.rs)

The GA:

- seeds the initial population from the saved pairwise top-10 expressions
- uses mutation, crossover, random injection, elitism, and tournament selection
- optimizes by `logloss`
- uses `ROC AUC`, height, and size as tie-breaks

## Command

```bash
cargo run --manifest-path rust/eml_tree_search/Cargo.toml --bin ga-best-expression -- --repo-root /home/tarstars/prj/titanic_mle --population 256 --generations 150 --max-height 5 --seed 20260420
```
